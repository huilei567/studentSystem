## 使用说明

1.  下载python11.0版本
2.  打开本项目地址，复制链接，下载到本地
3.  右键app.py,运行项目
4.  默认账号和密码：
~~~python
admin
~~~
当然，你也可以在login文件下更改大概在第28行
